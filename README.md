## Task Performance Dashboard

A dashboard for tracking daily and weekly task performance with:
- Daily view with task status distribution chart
- Weekly view with productivity trend analysis
- Task list with status and priority indicators
- Responsive design with Tailwind CSS

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Pages

- `/` - Home page with overview stats
- `/dashboard/daily` - Daily task performance view
- `/dashboard/weekly` - Weekly performance trend view
